# App Icons

Placeholder directory for application icons.

To generate icons, run:
  npx @tauri-apps/cli icon

This will generate 32x32.png, 128x128.png, 128x128@2x.png, icon.icns, and icon.ico
from a source image (default: app-icon.png in the project root).

Required icons (as configured in tauri.conf.json):
- icons/32x32.png
- icons/128x128.png
- icons/128x128@2x.png
- icons/icon.icns
- icons/icon.ico
